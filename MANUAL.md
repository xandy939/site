# Manual — By TR Alojamentos

## Entrar no painel
1. www.apartamentosturisticosalgarve.pt → Login
2. Perfil → botão **🛠️ Painel de Administração**

## Confirmar reserva por IBAN
1. Recebes email "💳 Nova reserva por transferência" com a referência `TR-XXXXXXXX`
2. Verificas no Bankinter que a transferência entrou com essa referência
3. No admin → encontras a reserva (estado "A pagar")
4. Botão verde **"Confirmar pagamento"**

*Reservas Multibanco confirmam-se sozinhas em 5-30 min, não precisam de nada.*

## Mudar preços
Admin → **Épocas e Tarifas** → escolher apartamento → **+ Nova época** ou Editar/Apagar na linha.
Datas sem época usam o preço base.

## Gerir clientes
Admin → **Clientes** → Ver reservas / Apagar conta.

## Booking.com — sincronização
Está configurada. Demora 15-60 min em cada sentido.
**⚠️ No Extranet, escolher SEMPRE "adicionar bloqueios" — NUNCA "espelhar calendário"** (espelhar apaga reservas reais).

## Problemas comuns
| Sintoma | Solução |
|---|---|
| Multibanco demora a confirmar | Esperar 30 min. Se passar, confirmar manualmente. |
| IBAN: cliente diz que pagou | Verificar Bankinter → confirmar no admin |
| Datas Booking não aparecem | Esperar 30 min |
| Cancelar reserva paga | Faz o reembolso pelo banco — o admin só liberta as datas |
| Email reset não chega | Pasta SPAM. Esperar 60s antes de tentar outra vez |
